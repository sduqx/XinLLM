# XinLLM - 从零构建大语言模型

基于《Build A Large Language Model》的学习实践项目，从零开始实现一个简单的大语言模型。

## 📚 项目目标

通过逐步实现以下组件，深入理解大语言模型的工作原理：

- 文本数据处理与Tokenization
- 注意力机制（Attention Mechanism）
- Transformer架构
- 模型训练与优化
- 文本生成

## 🚀 快速开始

### 环境要求

- Python 3.8+
- PyTorch 2.0+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行示例

```bash
python main.py
```

## 📖 学习路径

本项目按照以下步骤逐步构建：

1. **数据准备** - 文本处理、Tokenization
2. **模型架构** - 实现Transformer核心组件
3. **训练流程** - 损失函数、优化器
4. **推理应用** - 文本生成

## 🛠️ 技术栈

- PyTorch - 深度学习框架
- NumPy - 数值计算
- tqdm - 进度条显示
